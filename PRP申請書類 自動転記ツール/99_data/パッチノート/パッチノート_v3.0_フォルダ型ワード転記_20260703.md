# 📋 PRP申請書類 自動転記ツール — パッチノート

## v3.0 （2026-07-03） — 2種/3種を「フォルダ型（Excel様式＋Word書類＋PDF同梱）」に対応

> テンプレートが **1つのExcelブック** から **フォルダ構成** に変わったことに合わせ、
> 提供計画（2種・3種）も SOP と同じ「フォルダごと出力」方式へバージョンアップしました。

---

## ✨ 主な変更

### 1. テンプレートの新構成に対応（フォルダ型）
各テンプレは「フォルダ」になり、中身は以下の3種類：

| 種別 | 例 | 処理 |
|---|---|---|
| **Excel（様式第一の二・1ページ目）** | `01.【様式第一の二】…(PRP二種）.xlsx` | セル転記（従来どおり） |
| **Word（その他の書類）** | `2 提供する…`, `8 概要書`, `9 標準書`, `説明書`, `略歴書`, `同意書`, `アセント` 等 | サンプル値をヒアリング値へ一括置換 |
| **PDF・論文・追加Excel** | `【PRP整形】提供医療機関一覧.pdf`, `6.整形2種論文/*.pdf`, `教育研修記録表フォーマットKH2025.xlsx` | **変更せず同梱**（作業者の参照用） |

### 2. 出力もフォルダごと
```
02_output/
├─ 2種関節系PRP_(ヒアリング名)/   … 様式xlsx＋Word書類＋PDF一式
├─ 3種筋腱靭帯系PRP_(ヒアリング名)/
└─ SOP_(ヒアリング名)/            … Word16本＋教育研修xlsx
```
- フォルダ名＝`(テンプレ名)_(ヒアリングシート名)`、**中のファイル名は変更なし**。

### 3. Word書類への転記（一括置換方式）
以下のスカラー項目を、ヒアリングの値でWord全書類に差し込みます（**図・表・書式は保持**）。
- クリニック名／住所／郵便番号／電話番号／メール
- 氏名（開設者・管理者・医師）
- 委員会名／認定番号／受付番号

**テキストボックス内**の文字も置換対象（低レベル走査を追加）。

## ⚠️ 今回の範囲と、次段階（重要）

「とりあえず一括置換で」の方針に沿い、**識別情報（スカラー値）を確実に転記**する版です。
以下は**未対応**で、テンプレの既定テキストのまま残ります（次段階＝プレースホルダ方式で対応予定）：

- **キット製造方法の本文**（採取／加工／投与）… テンプレは Condensia＋Mycells のまま。
  実際に選択されたキット（例：Condensia＋GPSⅢ＋APS）への差し替え・①②③連番展開は次段階。
- 略歴書などの**詳細プロフィール文**（氏名以外の経歴等）。
- 役割別の氏名の厳密な打ち分け（管理者／施設管理者の別）… 現在は全て開設者名で統一。
