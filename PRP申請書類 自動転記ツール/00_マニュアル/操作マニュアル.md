# PRP申請書類 自動転記ツール ― 操作マニュアル

ヒアリングシート（PRP）から、申請書類一式（2種関節系／3種筋腱靭帯系）を
**全シートまとめて**自動転記するツールです。

---

## いちばん簡単な使い方（3ステップ）

1. **`01_input【ヒアリングシートをここへ】`** に、記入済みの
   `PRPヒアリングシート.xlsx` を入れる（ファイル名は自由）。
2. ルートの **`転記実行.bat`** をダブルクリック。
3. **`02_output【転記済みファイルがここに生成】`** に、書類が3ファイルできます。
   - `2種関節系PRP_◯◯_日時.xlsx`（11シート：様式第一の二・書類2・医師略歴書・患者説明書 ほか）
   - `3種筋腱靭帯系PRP_◯◯_日時.xlsx`（同上）
   - `SOP_◯◯_日時.xlsx`（17シート：衛生／製造／品質管理基準書＋各手順書。図面・組織図の画像も保持）

   ログは **`03_logs`** に出ます。**黄色の行（確認対象／未割当）は必ず目視確認**してください。

> テンプレートは `99_data/テンプレート/` に3つ（2種・3種・SOP）。増減すると出力数も変わります。

> 元のテンプレート・原本は上書きされません（毎回別名で出力）。

---

## フォルダ構成

| フォルダ／ファイル | 役割 |
|---|---|
| `転記実行.bat` | これをダブルクリックで実行 |
| `00_仕様書・マニュアル` | 本書など |
| `01_input【ヒアリングシートをここへ】` | ヒアリングシートを置く |
| `02_output【転記済みファイルがここに生成】` | 完成書類（全シート保持） |
| `03_logs` | 転記ログ |
| `99_data` | 内部システム（触らない） |
| └ `テンプレート` | 2種／3種テンプレ(.xlsx) |
| └ `マッピング` | `mapping.json`（対応表）・`format_changes.json`（様式変更用） |
| └ `src` | プログラム（transcribe.py / phase1 / launcher.bas） |
| └ `参考元【原本・設計書】` | 原本ヒアリング・データマッピング・設計書 |

## 何が転記されるか

- **様式第一の二（提供計画）**：医療機関名・住所・管理者・再生医療等の名称・実施責任者・
  事務担当者・医師・救急設備・PRP加工方法・細胞加工施設・認定委員会（認定番号）
- **医師略歴書**：ヒアリングの「略歴書（PRP）」タブから氏名・生年月日・所属・役職・専門分野ほか
- **書類2（詳細）**：PRPキットの採取方法・加工方法（ヒアリングの「PRPメーカー」タブから）
- **患者説明書／概要書／標準書／同意書／アセント文書**：医療機関名・住所・管理者・委員会 ほか
- **SOP（衛生／製造／品質管理基準書）**：各基準書 冒頭の医療機関名（手順書D〜Pは汎用のため対象外）

> シートを跨いだ取得（クロスシート検索）に対応：選択したPRPキット名や委員会名をキーに、
> 別タブ（PRPメーカー・略歴書・審査委員会）から詳細を自動で引いてきます。

---

## 確認対象（黄色）について

自動で入れたが**人の確認が必要**な項目はログで黄色になります。主なもの：
- 管理者氏名（医療法人＝役職+氏名／個人＝氏名のみ）
- 実施責任者・事務担当者の所属部署（ヒアリングに欄がなく医療機関名で代用）
- 事務担当者FAX（ヒアリングに欄なし）／医師役職（既定「院長」）
- PRPキットの採取・加工方法（PRPメーカー由来。体裁の微調整が要る場合あり）
- 「未割当（要マッピング）」＝緑セルだが転記元が未定義（書類2の空セル等を含む）

---

## ヒアリングの行を消しても動く？

→ **動きます。** セル位置ではなく「確認事項の文言（ラベル）」で探すため、行の増減に強い設計です。

---

## 様式・項目名が変わったら（メンテ担当向け）

`99_data/マッピング/format_changes.json` に変更を書くと、コードを直さずに追従できます
（項目名の改称・セル移動・転記元の差し替え・項目の無効化）。詳細は
`99_data/src/README_メンテナンス.md` を参照。

---

## うまく動かないとき

- 「openpyxl が無い」→ コマンドで `py -m pip install openpyxl`
- 「mapping.json が無い」→ `py 99_data/src/phase1_build_template.py` を実行して再生成
- ログが文字化け → Excelファイル（03_logs内）を直接開く
